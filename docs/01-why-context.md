# 为什么需要 Context 管理

## 没有 Context 管理的痛苦

你有没有遇到过这些情况：

- 每次跟 AI 对话，都要重新介绍自己
- 上次聊过的内容，AI 完全不记得
- 你说过的偏好、目标，AI 每次都要再问一遍

**这就是"没有记忆"的 AI。**

## Context 是什么？

**Context = 你想让 AI 知道的关于你的一切**

- 你的背景、目标、偏好
- 你做过什么、学过什么
- 你怎么思考、怎么决策
- 你的知识和方法论

## 有了 Context 管理

- AI 记住你是谁、想做什么
- 对话越来越高效
- 你的知识库持续积累
- AI 从"陌生人"变成"懂你的伙伴"

## 这个教程帮你做什么

不是教你"Context 是什么"（你现在知道了）

而是教你**一套工具和方法**：
- 用 GitHub 私有仓库存储
- 让 AI 自动整理和 commit
- 跨平台、跨设备调用

**一格一格来，慢慢积累。**
