消除 AI 孤岛，让所有 Agent 共享同一个大脑。

支持自动总结对话、跨 Agent 异步同步，通过 Git 持久化自己的偏好与知识，实现 Agent 间的无缝认知传递。 

通过 Git 协议，实现真正的 Personal Knowledge as a Service (PKaaS)。
